# v0.21.0 — Gate C evaluator

Gate C scores sanitized `AgentTrace` JSON. It never reconstructs or requests
hidden reasoning. Dependency success is earned only when every required tool is
present in a valid order and each later input carries the exact ID returned by
the preceding output.

Replay fixtures are committed, deterministic, and safe for CI. Live mode checks
the resolved provider and model before constructing any request: only
`openai / gpt-5.6-luna-xhigh` is admissible; Sol, Terra, OpenRouter, or another
slug is an immediate non-network failure.

The committed replay passes all frozen metrics. The 2026-07-17 live gate remains
incomplete because the exact Luna slug was unavailable at the approved endpoint;
no replay result may be presented as a live Gate C pass.
