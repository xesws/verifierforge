# v0.21.0 — Agent admission gate

This gate is separate from model-training Gates A/B. It evaluates Forge Agent
decision behavior and never starts a model server, trainer, GPU, or provisioner.
Thresholds are compiled constants and cannot be relaxed after observing a run.

An annotated `agent-gate-c-pass` tag is legal only after a complete Luna live
run passes all four thresholds. Replay success alone is regression evidence,
not product admission.
