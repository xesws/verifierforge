# v0.13.0 — Synthetic product traffic source

This D5 input generator consumes, but does not rewrite, the frozen
`data/nl2sql/v0.10.0-training-pool.jsonl` prompts. It combines those SQL data
pull requests with self-authored support-ticket and invoice extraction requests.
Each family has a distinct system prompt so the proxy's SHA-256 key produces
three deterministic clusters for tomorrow's Discover work.

`scripts/traffic_gen.py` sends to the local proxy at a configurable rate and
stops cleanly on interruption after printing sent/success/failure totals.
`scripts/traffic_summary.py` groups persisted metadata by system-prompt hash and
prints request count, aggregate tokens, and estimated cost. Neither script is an
evaluation, gate, or training input; D3/D4 frozen artifacts stay unchanged.
