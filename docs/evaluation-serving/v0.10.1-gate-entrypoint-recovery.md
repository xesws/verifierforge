# v0.10.1 — U2 rerun and held-out before snapshot

The repaired invocation runs the 50-row v0.10.0 training pool at `k=8` with
sample evidence. Admission still requires all fixed predicates:
`0.20 <= pass@1 <= 0.60`, `mixed_fraction >= 0.30`, and `pass@8 >= 0.85`.

After that completed evaluation, the 60-row held-out set also runs at `k=8`
with saved samples in `--reference` mode. Its triplet is stored as the D4
pre-training baseline only; it cannot make a failing training-pool Gate A pass
or be used for training monitoring. Both runs bind input, verifier, model, and
sample hashes in evidence.
