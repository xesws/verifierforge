# v0.10.2 — evidence bound to the U3 freeze

The manifest binds the completed U2 training-pool evidence and sample artifact
to the 50-row SHA-256, and binds the completed held-out reference baseline and
sample artifact to the 60-row SHA-256. Both must agree with verifier v2 source
provenance and the same pod-local model/config identity.

The training-pool gate result is an admission record only. The held-out
reference triplet from v0.10.1 is the exclusive pre-training baseline for D4,
post-training comparison, and DoD statements.
