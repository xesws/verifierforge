# v0.15.2 — real endpoint evaluation handoff

The step-350 export remains the canonical `7bde…` tree established by M6 and
v0.15.1. Dependency alignment changes only the serving runtime; it cannot
change the held-out result (`0.7833 / 0.9000 / 0.4333`).

If the local and public serving gates pass, the guardian samples the real
endpoint's data-pull SQL completions at the existing configured rate. The
temporary 50% canary is validated from persisted route rows and live-pass
points, then returned to zero. A guardian failure never blocks proxy traffic;
it is recorded as sidecar evidence.
