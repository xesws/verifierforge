# v0.8.0 — evaluation evidence for verifier v2

Gate A sample evidence remains append-only and retains the raw model
`completion`. v2 adds an additive extraction object: whether normalization was
applied, its kind, and the exact `scored_completion` that entered the unchanged
tier scorer. Aggregate Gate A evidence records verifier version `2` alongside
the existing source SHA-256; no fields consumed by the freeze helper are
renamed or removed.

Before a live rerun, the offline A2 verifier command writes an atomic JSON
artifact for the 276 stored candidate `reference_sql` values. A non-full score
is a hard stop, not a reason to alter the candidate dataset or score threshold.

## A3 result

The v2 subset evaluation completed all 400 samples with retained raw and
scored-completion evidence. Its result was `0.64 / 0.82 / 0.42` for pass@1,
pass@8, and mixed fraction. It is a completed Gate A rejection because pass@1
exceeds the fixed `0.60` maximum; the evidence remains an audit record and does
not permit a threshold adjustment.
