# v0.22.5 — Public endpoint proof

The endpoint proof first discovers the served model through `/v1/models`, then
sends one deterministic NL-to-SQL completion using the official OpenAI SDK.
The client sets `max_retries=0`, bounds its timeout, emits sanitized JSON, and
never prints request headers or the API key.

Loopback failure sends no public request. A timeout, authentication error,
non-JSON response, empty completion, or non-2xx public response is preserved as
evidence and stops the serving lane. Only a passing public request can unlock
the real tuned canary and guardian proof.
