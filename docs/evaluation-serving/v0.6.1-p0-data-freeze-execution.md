# v0.6.1 — Gate A execution evidence

**Status:** in progress. Linked plan: `docs/versions/v0.6.x/v0.6.1-p0-data-freeze-execution.md`.

Gate A runs at `k=8` on both frozen inputs. The full candidate set is a reference measurement; only the deterministic 50-row subset controls admission. The subset passes exactly when `0.20 <= pass_at_1 <= 0.60` and `mixed_fraction >= 0.30`; no code or operator may adjust those values tonight.

Evidence records input and verifier hashes, `pass_at_1`, `pass_at_8`, `mixed_fraction`, sampling parameters, model identifier, and a redacted base endpoint identity. It contains no credentials, prompts, completions, URL paths, or query strings. Both measurements finish before the subset decision is acted upon, so a rejected subset still has its required full-set reference figures.
