# v0.15.0 — Real-endpoint guardian check

The M6-selected, standard-HF step-350 export is the only model eligible for
this serving proof. The public endpoint is first tested with a single
OpenAI-compatible NL-to-SQL completion. Then the existing D5 proxy is pointed
at that endpoint only through `VF_PROXY_TUNED_UPSTREAM` and runs a 50% canary
for `data-pull-sql` traffic.

The existing asynchronous guardian remains unchanged: it samples tuned SQL
responses and scores them with `NL2SQLVerifier`. Its observed rolling
`LivePassRate` is deployment telemetry, not a new held-out benchmark. The
acceptance floor is 0.5; a lower real value stops the rollout and leaves
canary at zero.
