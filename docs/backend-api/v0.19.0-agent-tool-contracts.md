# v0.19.0 — Agent tool boundary

A-1 adds no HTTP routes. It establishes typed tool interfaces that a later API
may invoke through the guarded Runner. Existing Job, Cluster, routing, guardian,
mock, and reviewer-artifact responses are unchanged.

The registry is an application boundary, not an execution surface. Its real
binding is read-only, its mock binding is deterministic, and neither exposes a
trainer or provisioner handle.
