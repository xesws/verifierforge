# v0.17.0 — read-only demo artifact API

`VF_API_DATA_MODE=artifacts` makes the real FastAPI service read committed
`data/demo-artifacts/` instead of ignored `runs/`. It preserves existing
`/jobs`, `/jobs/{id}`, `/jobs/{id}/metrics`, routing, and live-pass response
shapes; no contract fields change. The mode is intentionally immutable: routing
PUT returns a clear read-only error rather than pretending to change a demo.

Artifacts are validated at load time as the existing Pydantic contracts, and
their metrics arrays are reconstructed from committed append-only JSONL. The
default remains local `runs/`, so developer and trainer workflows do not change.
