# v0.34.1 Registry-only tuned routing

**Status:** in progress

Production proxy construction ignores and no longer parses static tuned URL,
key, or model environment variables. A logical `tuned` route resolves only a
ready `serving_endpoints` record. Cold/non-ready state uses the default upstream
and returns an observable fallback route header without changing the frozen
OpenAI-compatible JSON response.

Explicitly injected proxy settings retain deterministic fake-tuned behavior for
offline tests; they cannot provide a physical HTTP endpoint.
