# v0.4.0 — Backend API: Copilot Proposal Routes

## Scope

Add additive, laptop-local routes:

- `POST /copilot/nl2sql/proposals` creates reviewable candidates from a task,
  schema, examples, and seed count.
- `POST /copilot/nl2sql/validate` validates a supplied candidate in the Docker
  sandbox and returns bounded diagnostics.

Existing `/jobs` and `/jobs/{id}/metrics` contracts remain unchanged. API request
and response models are local to the API module so shared frontend contracts do
not change prematurely.

## Validation

Use a fake OpenRouter client and fake sandbox in unit tests. Validate malformed
structured output as a clear 502-style provider failure, never a silent fallback.
