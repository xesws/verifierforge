# v0.22.0 — Lightweight Discover demo

Because no frontend source tree exists in this repository, FastAPI serves one
dependency-free Discover page. It renders the three stable product clusters,
calls the real agent API, shows decisions and rationale, conditionally shows a
training config, and posts the idempotent approval. It is deliberately a demo
surface, not a new frontend framework.

The page is unavailable while the flag is off. A successful local mock-binding
walkthrough is captured under `assets/forge-agent/`; the screenshot cannot be
used to claim Gate C live admission.

The captured page shows the deterministic `data-pull-sql` recommendation,
auditable training config, cache marker, and completed human approval. Screenshot
