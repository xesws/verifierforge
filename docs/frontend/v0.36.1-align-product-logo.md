# v0.36.1 frontend brand alignment

**Status:** complete

The `/tech` wordmark and browser favicon must visually match `.brand-mark` in
the reviewer product: rounded blue/green gradient tile, white `V` in the upper
left, white `F` in the lower right, and the same subtle diagonal separator.
The SVG generator is the shared implementation source so README, `/tech`, and
favicon cannot drift independently.

`FloatingNav` and `InviteGate` now render `/favicon.svg` instead of rebuilding
the mark with CSS children. The same generator emits the favicon and `/tech`
wordmark, making the visual identity byte-stable across those surfaces.
