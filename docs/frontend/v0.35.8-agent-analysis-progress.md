# v0.35.8 — Discover analysis progress

The Discover Analyze action will expose an in-place live activity panel as
soon as the request is sent. It shows elapsed time, a completed request stage,
an active bounded-Agent stage, and a pending validated-receipt stage.

The API is currently a synchronous request rather than a streaming event feed,
so the UI deliberately uses an indeterminate activity indicator. Exact tool
completion remains visible only after the server returns the persisted Agent
run receipt. This avoids presenting a timer-driven guess as real backend
progress while still assuring reviewers that the request is alive.

The status panel uses `role="status"` and `aria-live="polite"`; the Analyze and
Input actions remain disabled until the request settles.
