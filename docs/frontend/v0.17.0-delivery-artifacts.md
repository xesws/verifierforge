# v0.17.0 — reviewer-safe data source

The frontend can point at the real API with `VF_API_DATA_MODE=artifacts` and
receive the same job, metric, routing, and live-pass shapes as development
mode. The shipped main job is the factual held-out selection at step 350:
pass@1 improved from `0.5833` to `0.7833`; the random-reward control is shipped
alongside it so a visual report does not imply causal proof from one curve.

Artifact mode is read-only. Controls may display their disabled state but must
not present a successful canary switch or public model endpoint when the real
proxy route is unavailable.
