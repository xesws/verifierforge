# v0.33.0 hosted frontend handoff

**Status:** in progress — 2026-07-20

The frontend will set `VITE_VF_API_BASE_URL` to the Railway public subdomain.
Product requests use the separately shared Basic Auth invitation; health is
public. CORS permits configured localhost origins, Vercel preview subdomains,
and the final Vercel URL without opening a wildcard origin.

The hosted service keeps `VF_AGENT_BINDING=mock` and
`VF_AUTOPROVISION=false`, so Analyze is deterministic and Start Forge returns
the documented disabled response. The frontend contract remains the 19
operations frozen by `frontend-api-v1.1`.
