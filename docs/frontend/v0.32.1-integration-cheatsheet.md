# v0.32.1 — Frontend integration cheatsheet delivery

This patch publishes `docs/frontend/integration-cheatsheet.md` as the owner
handoff for the real Postgres API. It will contain the exact launch command,
loopback base URL, per-action headers, current feature-flag values, disabled
semantics, CORS configuration, and a 16-row result/provenance table with
failures first.

The handoff must distinguish shipped facts from contract aspirations. In
particular, immutable D4 artifacts, local run files, static cluster catalog,
and Supabase enrichment are named separately. Missing arena/savings data or a
mixed source is recorded as an integration gap; this documentation patch will
not fabricate frontend data to make a check pass.
