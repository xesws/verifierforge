# v0.35.7 — Discover Agent run receipt

Discover will show the actual validated Agent record beneath the proposal:
provider, model, trace ID, timestamps, token counts, cache status, read-only
tool calls and outputs, and the terminal structured decision. Tool details are
expandable; the final decision JSON remains directly visible.

This panel is an audit transcript, not hidden chain-of-thought. Mock and cached
runs are explicitly labelled and never presented as a fresh live-model result.

Production acceptance displayed a fresh `openai / gpt-5.6-luna` receipt with a
unique trace ID, four expandable tool records, token totals, and the exact
validated `forge` JSON. Reloading the decision correctly changed only the cache
label while preserving trace identity.
