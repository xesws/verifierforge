# v0.40.0 — `/tech` print/export mode

The browser article remains the only rendered content implementation. A
`?print=1` mode makes images eager, removes interactive-only controls, opens
all 12 chapter panels, and applies A4 print rules with deliberate chapter
transitions and print-safe typography.

Normal `/tech` layout and interaction remain unchanged. The exporter consumes
the production build locally, so the submitted PDF cannot drift from the
versioned article or its six generated SVG figures.
