# v0.35.6 — Schema-grounded SQL generation

The Ship generator and browser SQLite executor now share a schema exported
from `data/nl2sql/v0.10.0-training-pool.jsonl`. Every tuned completion receives
the exact table and column definitions before the reviewer's question.

The schema prompt contains DDL only, not fixture INSERT rows or reference SQL.
SQLite execution remains independent and still surfaces genuine model errors;
the UI does not substitute a canned query or result.
