# v0.35.2 — Ship prompt examples

The tuned playground presents six one-click examples from the same frozen
employee/department/project schema used for the NL→SQL forge. Examples cover:

- a single-table maximum;
- a filtered and ordered project list;
- an employee/department join;
- grouped average salary with a threshold;
- project-hour aggregation across the assignment table;
- highest-paid employee per department.

Selecting an example only fills the textarea. It never submits a request,
wakes a model, or changes canary routing. The reviewer may edit the populated
question before explicitly pressing **Run tuned completion**. The selected
chip exposes `aria-pressed=true`, and the controls remain disabled while a
completion is running.

The deployed selector contains all six labels and replaced the prior
customer-order default, which was outside the frozen schema.
