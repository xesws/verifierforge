# v0.4.5 — Model Trainer: Use verl's Standard PPO Entry Point

D2 now invokes `verl.trainer.main_ppo` rather than `main_ppo_sync`. Both use
the same D2 overrides and native checkpoint layout, but the standard trainer
correctly assigns the reference policy to the LoRA-enabled actor worker.

This is a targeted compatibility correction for the pinned verl 0.8.0 runtime;
it intentionally preserves the D2 LoRA rank, KL settings, and Storage bridge.
