# v0.4.6 — Model Trainer: DDL-Only NL2SQL Prompts

The D2 case payload now has two schema representations:

- `prompt` contains compact DDL so the model can identify tables, columns, and
  joins within the 1024-token rollout budget.
- `schema_sql` retains DDL plus deterministic `INSERT` fixtures for the
  programmatic verifier.

This keeps training context small without weakening reward correctness or
changing the reviewed 40/10 split.
