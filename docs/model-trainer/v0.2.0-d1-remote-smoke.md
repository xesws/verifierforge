# v0.2.0 — Model Trainer: D1 Remote Smoke

## Planned Change

Use only the GPU-free fake trainer to verify launch, metrics append, checkpoint creation, resume, and final artifact storage on the remote worker. The `fake_smoke` configuration runs 150 steps at two seconds per step so the detached process remains observable for roughly five minutes. No real model, GRPO, verl, or vLLM dependency belongs in this version.

## Acceptance Evidence

Record the fake-job arguments, tmux session name, resumed step, metrics JSONL line count, checkpoint path, and final artifact path. The worker must continue after the SSH command returns.
