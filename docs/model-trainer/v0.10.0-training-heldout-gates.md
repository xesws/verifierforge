# v0.10.0 — training versus held-out data ownership

The 50-row training pool is the sole data source for Gate B and D4 GRPO. The
60-row held-out set is never passed to the trainer and is reserved for baseline,
after-training, and DoD evaluation. Both artifacts are generated before freeze,
but neither replaces `trainer/data/nl2sql_v1.jsonl` until the new Gate A passes
and a three-piece freeze is committed.

The promotion manifest must name the training pool, held-out pool, verifier v2,
and the deterministic split rule. Training logs may show training-pool reward or
pass curves, but no such curve is a held-out performance claim.
