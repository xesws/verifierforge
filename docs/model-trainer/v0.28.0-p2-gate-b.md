# v0.28.0 — P2 0.5B S3 training profile

`grpo_v1_0p5b_p2` is a configuration-only addition. It uses the frozen
`v0.10.0` 50-row training pool, Qwen2.5-0.5B-Instruct, 100 total steps, k=8,
checkpoint interval 50, verifier reward and serviceable HF export. No training
algorithm or frozen-data byte changes are allowed.

The job runs with `VF_STORAGE_BACKEND=s3` on a pod without a persistent volume.
The executor declares completion only after S3 exposes step-100 metrics,
checkpoint publication metadata, `artifacts/final/model.txt`, and
`artifacts/curve.png`, with SHA-256 recorded for every collected object.
