# v0.4.7 — Model Trainer: Include the NL2SQL Verifier Dependency

The D2 custom reward worker imports the shared `NL2SQLVerifier`, which uses
`sqlparse` for parse-tier scoring. `requirements-trainer.txt` now declares
`sqlparse>=0.5` independently of the app requirements so a GPU-only pod can
load the verifier without relying on the API environment.
