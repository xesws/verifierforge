# v0.4.3 — Model Trainer: Use PyTorch SDPA for D2

The D2 L4 environment intentionally uses the already-proven torch
2.8.0+cu128 / verl 0.8.0 / vLLM 0.10.2 stack. Because that environment does
not include the optional FlashAttention2 package, the GRPO command explicitly
sets `actor_rollout_ref.model.override_config.attn_implementation=sdpa`.

This is a compatibility choice for the bounded smoke run. It prevents verl's
default `flash_attention_2` selection from failing model construction; it is
not a performance tuning decision or a commitment for future larger runs.
