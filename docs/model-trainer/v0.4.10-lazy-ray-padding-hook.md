# v0.4.10 — Model Trainer: Lazy Compatibility Injection for Ray

The SDPA padding fallback belongs to the trainer worker, not Ray's dashboard or
runtime-environment agents. v0.4.10 registers a lightweight import hook at
process startup and delays the verl/Transformers imports until
`verl.utils.attention_utils` is loaded. This preserves the v0.4.9 helper
resolution while keeping raylet's metrics-agent startup budget clear.
