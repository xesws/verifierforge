# v0.4.4 — Model Trainer: Add the SDPA Map Key Correctly

verl's base Hydra configuration declares `override_config: {}` as a structured
empty mapping. Adding `attn_implementation` therefore requires the command
override `+actor_rollout_ref.model.override_config.attn_implementation=sdpa`.

The leading `+` is configuration syntax, not part of the key. It preserves the
v0.4.3 decision to use torch SDPA on the D2 pod and allows the setting to reach
verl's model constructor.
