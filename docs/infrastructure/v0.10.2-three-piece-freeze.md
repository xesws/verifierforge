# v0.10.2 — immutable tag and detached Gate B procedure

The U3 manifest is generated locally from the frozen data/evidence, committed,
and then annotated as `v0.10.2-p0-three-piece-freeze`. The tag annotation names
the evaluated local model but contains no credential. `main` and the tag must
both reach origin before RunPod starts work.

The pod checks out the exact tag commit, then stops `vf-eval-vllm` and verifies
that its vLLM process/GPU allocation is gone before `vf train` detaches Gate B.
The existing `vf` preflight import remains mandatory. Poll the detached job no
more often than every 120 seconds; preserve logs/artifacts on any failure.
