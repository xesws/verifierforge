# v0.17.0 — serving and reviewer packaging

`requirements-serve.txt` locks the verified direct serving dependencies,
including vLLM 0.10.2 and the compatible Transformers 4.57.6 stack that fixed
the tokenizer failure. It is a serving/proxy environment, not a trainer
environment. The endpoint key remains an ignored environment variable.

The repository may record the endpoint's local verification configuration and
the public proxy blocker, but never a secret or an invented price/availability
claim. A reviewer can run the artifact API without a GPU or cloud account.
