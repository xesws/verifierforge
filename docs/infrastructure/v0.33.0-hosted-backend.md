# v0.33.0 Railway reviewer hosting

**Status:** in progress — 2026-07-20

Railway Hobby is the selected host. Deploy exactly one production service,
disable Serverless sleep, use the Railway-provided fixed subdomain, and set a
usage alert within the plan's monthly credit. No duplicate staging service is
allowed.

All database, S3, invitation, encryption, tuned-endpoint, and CORS values are
platform secrets. OpenAI/OpenRouter/RunPod provisioning keys are deliberately
absent. Deployment runs Alembic and the deterministic flagship report
projection sync before one non-root Uvicorn worker starts.

The old laptop sandbox and GPU-side non-vLLM processes may be stopped only
after the public contract/auth/CORS/proxy matrix passes. If RunPod port 8000
cannot be exposed, the quick-tunnel rotation runbook changes only
`VF_PROXY_TUNED_UPSTREAM`; report pages continue in artifact fallback mode.

## Hosted environment inventory

Railway sets `PORT`. Required application secrets/settings are:

- Access and database: `VF_REVIEW_INVITE_CODE`, `VF_DB_BACKEND=postgres`,
  `SUPABASE_DB_URL`, `VF_CRED_KEY`.
- Data and feature safety: `VF_API_DATA_MODE=hybrid`,
  `VF_AGENT_ENABLED=true`, `VF_AGENT_BINDING=mock`,
  `VF_AUTOPROVISION=false`, `VF_PROVISION_BINDING=mock`.
- Evidence: `VF_S3_BUCKET`, optional `VF_S3_PREFIX`, `VF_S3_REGION` or
  `AWS_DEFAULT_REGION`, `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, and
  optional `AWS_SESSION_TOKEN`.
- Proxy/Guardian: `VF_PROXY_UPSTREAM=fake`, `VF_PROXY_TUNED_UPSTREAM`,
  `VF_PROXY_TUNED_API_KEY`, `VF_PROXY_TUNED_MODEL` or `VF_ENDPOINT_MODEL`,
  optional Guardian sampling/window/pool variables.
- Browser: `VF_CORS_ORIGINS` for exact local/final origins and
  `VF_CORS_ORIGIN_REGEX=^https://[a-z0-9-]+\.vercel\.app$` for previews.

Do not set OpenAI, OpenRouter, or `RUNPOD_API_KEY` on the hosted reviewer.

## Local H1 evidence

The Python 3.11 non-root image is `98,709,804` bytes and loads zero torch,
vLLM, verl, Ray, or Transformers modules. A local container connected to real
Supabase and the existing tuned endpoint: all 19 frontend operations returned
their expected status, localhost/Vercel CORS passed at both API and proxy
mounts, one real tuned completion returned 200, and Guardian reached 129
points. The flagship report returned 400/200 curve points, ten arena cards,
`$3,850` savings, and `real_gain`. An unreachable endpoint produced a 200 fake
fallback, sanitized degraded health, and the same complete artifact report.
