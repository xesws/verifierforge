# v0.32.2 run artifact publication

This patch force-adds the reviewed current contents of `runs/` and `models/`
without weakening their global ignore rules. Future generated files therefore
remain ignored and require a fresh review before publication.

Ordinary Git stores only files below GitHub's 100 MB per-file limit. The two
step-350 `.safetensors` shards are excluded; existing file-level and tree
manifests preserve their recorded identity and location history. The sandbox
invite code is excluded as a credential.
