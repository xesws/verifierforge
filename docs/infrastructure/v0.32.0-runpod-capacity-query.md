# v0.32.0 — RunPod inventory and price preflight

Source of truth is the official GraphQL endpoint
`https://api.runpod.io/graphql`, authenticated with the same ephemeral Bearer
credential resolver as Pod REST calls. No response or error may include that
credential. The query requests only candidate IDs and capacity/price fields;
it does not constrain data center.

The 2026-07-19 read-only probe returned live lowest on-demand offers of
`$0.20/hr` RTX 4000 Ada Community (`Low`), `$0.24/hr` RTX 2000 Ada Secure
(`Low`), `$0.39/hr` L4 Secure (`Low`) and `$0.44/hr` A40 Secure (`High`). These
are evidence, not configuration: price and stock are queried again for every
provision. The live smoke has an independent `$1` ceiling and must prove Pod
absence after DELETE.
