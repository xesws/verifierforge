# v0.35.7 — Reviewer Agent binding disclosure

The hosted reviewer UI must disclose whether Analyze used `mock` or a real LLM
provider. The existing public Railway sandbox is known to have been designed
with `VF_AGENT_BINDING=mock`; it must not be marketed as a live model run.

Production may switch to `real` only when the Gate C receipt and required
provider credentials already exist. No secret value may appear in deployment
logs, trace payloads, UI metadata, or commits.
