# v0.32.1 — Backend integration self-check

**Status:** planned — 2026-07-19  
**Owner/worktree:** Codex on `main`  
**Release type:** patch integration/documentation hardening

## Problem and scope

The frozen frontend API needs one real Postgres-backed exercise before owner
integration. This patch will run the 16 operations frozen by
`tests/test_frontend_api_v1.py`, validate response fields, state each response's
actual provenance, and publish a concise frontend cheatsheet. It will also
replace the API's unconditional CORS wildcard with an environment-controlled
local-development allowlist and make the default-off Start Forge response
explicit without enabling provisioner spend.

## Non-goals

No trainer, verifier, frozen dataset, public contract field, report fabrication,
provider provisioning, GPU action, paid LLM request, database migration, or
feature-flag default change. Existing data gaps are reported rather than hidden
or repaired in this patch.

## Affected areas and contracts

- `app/api/` and `mock/`: CORS configuration only, plus the disabled Start
  Forge error detail; successful request/response contracts remain unchanged.
- `tests/`: local-origin, hostile-origin, environment override, disabled Start,
  and frozen-operation regression coverage.
- `docs/backend-api/`, `docs/database/`, and `docs/frontend/`: executable
  launch and provenance evidence.

The additive environment interface is `VF_CORS_ORIGINS`, a comma-separated
allowlist. When unset, only localhost and 127.0.0.1 on ports 3000, 5173, and
8080 are allowed. A wildcard requires the explicit value `*`.

## Validation and stop conditions

Run focused and full pytest, the Postgres repository contract suite, secret
scan, diff check, real API calls, and CORS preflights. The live self-check uses
mock Agent and disabled autoprovision, creates only bounded test rows in
Supabase, restores mutable product state, and removes test rows afterward.
Stop rather than expose a credential, enable provider spend, or mislabel
artifact/mixed data as Supabase.
