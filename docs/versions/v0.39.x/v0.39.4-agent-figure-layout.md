## Problem and scope

the strict `submit_decision` box crosses the Gate C Evaluator boundary, while
the Gate C result badge appears on the human approval/Provisioner side. The
figure therefore weakens the intended separation between evaluating the Agent
and authorizing paid execution.

Agent, four read-only tools, data layer, strict terminal decision, and Gate C
result. The human side will show only the explicit `Approve → Start Forge`
sequence, separated by the spending wall.
