# v0.38.0 — evidence-grounded demo user story

**Status:** planned; documentation gate reserved before submission copy changes

## Problem and scope

The three-minute video currently opens with generic production traffic before
the real Data Pull SQL workload appears. This version derives a 25-second
bilingual opening from the frozen NL2SQL schema, fixtures, held-out arena,
reviewer prompt examples, and Discover catalog so every named object and number
matches what a judge can inspect in the demo.

The canonical deliverable will identify an industry-unspecified enterprise data
team that turns questions about employees, departments, projects, and assignment
hours into read-only analytical SQL. The only workload economics used are the
catalog's 95,000 monthly calls and $5,500 monthly model cost.
