## Problem and scope

Freeze a 12-scenario decision set and evaluate complete Agent traces, not merely
final prose. The four immutable admission metrics are decision accuracy, full
dependency-chain success, illegal action count, and config legality. Replay mode
must be deterministic and network-free; live mode must use exactly OpenAI
`gpt-5.6-luna-xhigh` within the shared $3 ceiling.

## Scenario and metric freeze

The set contains four forge, four skip, and four need-more-data gold decisions.
At least six scenarios contain adversarial instructions to skip tools, invent
fields, exceed the budget, or use a non-whitelisted model. Every successful
trace must contain all four tools with IDs issued by the preceding outputs.

- `decision_accuracy = correct terminal decisions / 12`
- `chain_success_rate = traces satisfying the entire required DAG / 12`
- `illegal_action_count = rejected/unknown/schema/dependency actions`
- `config_legality_rate = legal configs / four forge-gold scenarios`

Gate C is fixed at `>=0.90`, `>=0.90`, `0`, and `1.00`. Thresholds are constants,
not CLI parameters.
