# v0.9.0 — verifier-v2 projection gate

The reprojection uses `NL2SQLVerifier.VERSION == 2` only; it does not modify
extraction or tier scoring. Every projected record's stored `reference_sql`
must independently score exactly `1.0` before the B3 model evaluation. The
projection report binds source population, B1 pass counts, output subset, and
verifier source/version hashes so a later freeze can distinguish score behavior
from difficulty selection.
