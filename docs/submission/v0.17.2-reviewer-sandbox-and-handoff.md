# v0.17.2 — reviewer sandbox and handoff

The reviewer path is deliberately local and reproducible: clone the private
repository, install the lightweight requirements, then launch the committed
artifact API and deterministic proxy. It demonstrates evidence delivery and
OpenAI-compatible proxy shape without implying that a public inference endpoint
or canary routing has been deployed.

The run-sheet morning report must distinguish passed work from timeboxed or
owner-action work. In particular, it must not describe the S3 native recovery
as a timed node-loss proof, and it must preserve the exact serving-public-gateway
timeout rather than masking it with a retry.
